#### 代码结构说明

cn.gudqs7.plugins 下有 3 个主要的包，即 `docer / error / search`，作用如下：
- docer：方法注释获取
- error：接入 IDEA 异常处理组件
- search：Search Everywhere 中 Api 这个 Tab 的接入及实现 (主要)
